# Minima PDF — App UI Kit

Interactive recreation of the core Minima PDF product surface: Library → Reader → Tool Drawer, plus the Settings dialog. Three-panel structure on the reader screen (Sidebar / Reader / Tool Drawer), per the Hermes WebUI-inspired layout.

Files: `Sidebar.jsx`, `LibraryGrid.jsx`, `ReaderView.jsx`, `ToolDrawer.jsx`, `SettingsPanel.jsx`, `data.js` (sample documents), `index.html` (click-through demo).

No source codebase or Figma was available for this product (see root readme) — this kit is built directly from the dark, low-glare color/type/motion spec provided.
