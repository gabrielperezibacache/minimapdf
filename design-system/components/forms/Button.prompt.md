Primary interactive action button, used for confirming, opening panels, and library actions.

```jsx
<Button variant="primary" onClick={() => {}}>Add document</Button>
<Button variant="secondary" size="sm">Cancel</Button>
```

Variants: `primary` (bronze fill), `secondary` (bordered), `ghost` (text-only), `danger` (bronze-red outline). Sizes: `sm`, `md`, `lg`. Hover states are flat color shifts, no shadow, transitions under 100ms.
